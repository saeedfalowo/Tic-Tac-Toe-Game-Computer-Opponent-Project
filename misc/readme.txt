
---------------------------------
HOW TO USE COMMAND PROMPT FOR C++
---------------------------------

Search and open Developer Command Prompt for VS on windows start menu

cd to the c++ file directory
Assume c++ file == hello.cpp

Enter:
	cl /EHsc hello.cpp

This compiles the c++ code and generates an object (.obj) file along with an executable
.exe file

Run the executable file hello.exe to run the code
